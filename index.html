Crystals Collector Game
Instructions

To start the game, you will be given a random number from 19 to 120. You are required to match this number, exactly.

Each of the four crystals is assigned assigned a random value from 1 to 12. As each selection is made, the value of the crystal is added to you total.

To win, you must match your number exactly, without going over.
